# Git Lab D02 

## To Remove Remote Branch
```
git push origin :branch_name
```

## To Remove Local Branch
```
git branch -d branch_name
```

## To List All Tags
```
git tag
```

## To Delete Local Tag
```
git tag -d tag_name
```

## To Delete Remote Tag
```
git push origin --delete tag_name
```
